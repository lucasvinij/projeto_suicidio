Projeto analisando dados sobre o suicídio

Esse dataset foi retirado de: https://www.kaggle.com/datasets/russellyates88/suicide-rates-overview-1985-to-2016 onde mostra mortes por suicídios de vários países de 1985 a 2016. Nesse projeto respondo algumas perguntas usando gráfico e código em R.

Ferramentas usadas: R,Tidyverse,Readr, Ggplot2 e Dplyr

Perguntas respondidas

  1. Quais gerações cometeram mais suicídios?
  2. Algum gênero cometeu mais suicídio que outro?
  3. Países com maior taxa de suicídio?
  4. Idades que mais cometeram suicídio?

# criando query sobre gerações com maiores taxas de suicídio
geracao <- df |> 
  group_by(generation) |> 
  summarize(
    total = n()
  ) |> 
  arrange(desc(total))

A geração X que vai de 1965 até 1981 junto com a geração silent entre 1925 e 1942 são as gerações que mais cometeram suicídio de todas as outras quatro gerações.

2.Mortes por gênero

Nosso dataset nos mostra apenas dois gêneros, os demais gêneros não constam ou não possuem dados suficiente.

# buscando todos os gêneros no dataset
df |> 
  select(sex) |> 
  distinct(sex)
##      sex
## 1   male
## 2 female
# agrupando dados e criando uma relação
generos <- df |> 
  group_by(sex) |> 
  summarize(
    n_mortes = round(mean(suicides.100k.pop, na.rm = TRUE),2)
  )

# chamando variavel
generos
## # A tibble: 2 × 2
##   sex    n_mortes
##   <chr>     <dbl>
## 1 female     5.39
## 2 male      20.2

Representando esse número em gráfico:

# criando gráfico
ggplot(generos, aes(x="", y=n_mortes, fill=sex)) +
  geom_bar(stat="identity", width=1, color="white") +
  coord_polar("y", start=0) +
  theme_void() + 
  labs(
    title = "Mortes por gênero"
  )

Com este gráfico podemos responder a segunda pergunta, o gênero masculino cometeu mais suicídios que o gênero feminino.

3.Mortes por país

# criando query e atribuindo a uma variavel
paises <- df |> 
  group_by(country) |> 
  summarize(
    mortes = sum(suicides_no, na.rm = TRUE)
  ) |> 
  arrange(desc(mortes))

# deixando apenas os 10 primeiros países
paises <- head(paises,10)

# plotando gráfico
paises |> 
  ggplot(aes(x = country,y = mortes, fill = country)) +
  geom_bar(stat = "identity") +
  coord_flip() +
  labs(
    title = "Suicídios por país",
    x = "Mortes",
    y = "Países"
  )

A Russia foi o país que mais cometeu suicídio, entre 1985 até 2016.

4.Idades que cometeram suicídio

# criando variavel com idade e sexo agrupados
df |> 
  group_by(age, sex) |> 
  summarize(
    mortes = sum(suicides_no, na.rm = TRUE)
  ) |> 
  arrange(desc(mortes)) |> 
  ggplot(aes(x = age, y = mortes, fill = sex)) +
  geom_bar(stat = "identity") # plotando o gráfico
## `summarise()` has grouped output by 'age'. You can override using the `.groups`
## argument.

O gráfico mostra que a maior taxa de mortes por suicídio acontece na classe de 35-54 anos, já a menor classe com taxa de suicídio seria de 5-14 anos de idade.
